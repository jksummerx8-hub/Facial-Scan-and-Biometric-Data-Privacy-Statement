臉部技術使用說明 TrueDepth API

當您使用「察覺 (Aware)」功能時，我們對您的數據處理承諾如下：

1. 我們收集什麼資料？

當您開啟「察覺」功能，Zenith 會透過您裝置上的 TrueDepth 前置相機系統 捕捉細微的面部表情（Blend Shapes）與頭部動作。

注意： 我們僅讀取數學運算所需的「特徵點數據」，絕不會拍攝、錄製或儲存您的照片或影片。

2. 這些資料用在哪裡？

這些數據僅用於即時計算您的「心能量」與「當下狀態」。透過分析面部肌肉的細微變化，App 能協助您更深層地了解自己的情緒起伏與疲勞程度，達成更好的靜心效果。

3. 資料會被存儲或分享嗎？

  • 100% 本地處理： 所有的運算都在您的 iPhone 或 iPad 晶片中即時完成。
  • 不儲存、不外流： 資料在運算結束後會立即銷毀。我們不會將您的臉部數據上傳至任何雲端伺服器，也不會與任何第三方分享。
  • 離線驗證： 您可以在開啟「飛航模式」的情況下使用此功能，證明數據完全不需要離開您的裝置。

4. 您擁有完全的控制權

您可以隨時在 iOS 的「設定」>「隱私權與安全性」>「相機」中撤回授權。若不開啟此權限，您仍可使用 Zenith 的其他所有功能。

5. 非醫療免責聲明
「察覺」功能所提供的數據僅供個人成長與靜心參考，不具備任何醫療診斷、預防或治療建議之效力。若您有健康方面的疑慮，請務必諮詢專業醫療人員。

隱私聲明 Privacy Policy

為了保障您的帳戶安全並提供更便捷的體驗，Zenith 提供臉部辨識功能。以下是我們的隱私保護承諾：

1. 權限啟動與請求

系統僅在您主動觸發功能時請求存取授權。未經操作，本程式不會擅自開啟相機或進行任何掃描。

2. 物理隔離運算

您的臉部辨識過程完全在個人裝置的硬體中完成。所有數據均不會上傳至伺服器或第三方雲端平台。

3. 不儲存原始影像

系統不儲存原始照片或錄影。掃描時會立即轉換為加密的數學特徵碼，且該代碼無法還原為人臉圖像。

4. 安全隔區保護

辨識數據儲存於裝置的 安全隔區 (Secure Enclave) 中，與作業系統其他部分隔離。當您卸載 App 時，相關數據將同步被永久刪除。
