臉部技術使用說明 TrueDepth API

當您使用「察覺 (Aware)」功能時，我們對您的數據處理承諾如下：

1. 我們收集什麼資料？

當您開啟「察覺」功能，Zenith 會透過您裝置上的 TrueDepth 前置相機系統 捕捉細微的面部表情（Blend Shapes）與頭部動作。

注意：我們僅讀取數學運算所需的「特徵點數據」，絕不會拍攝、錄製或儲存您的照片或影片。

2. 這些資料用在哪裡？

這些數據僅用於即時計算您的「心能量」與「當下狀態」。透過分析面部肌肉的細微變化，App 能協助您更深層地了解自己的情緒起伏與疲勞程度，達成更好的靜心效果。

3. 資料會被存儲或分享嗎？

  • 完全在本地處理： 所有的運算都在您的 iPhone 或 iPad 晶片中即時完成。
  • 不儲存、不外流： 資料在運算結束後會立即銷毀。我們不會將您的臉部數據上傳至任何雲端伺服器，也不會與任何第三方（如廣告商或分析機構）分享。
  • 離線驗證： 您可以在完全斷開網路（開啟飛航模式）的情況下使用此功能，這證明了數據完全不需要離開您的裝置。

4. 您擁有完全的控制權

您可以隨時在 iOS 的「設定」>「隱私權與安全性」>「相機」中撤回對 Zenith 的存取授權。若不開啟此權限，您仍可使用 Zenith 的其他冥想與音效功能。

5. 非醫療免責聲明
「察覺」功能所提供的數據僅供個人成長與靜心參考，不具備任何醫療診斷、預防或治療建議之效力。若您有健康方面的疑慮，請務必諮詢專業醫療人員。

隱私聲明 Privacy Policy

為了保障您的帳戶安全並提供更便捷的體驗，本服務提供臉部辨識功能。在使用此功能前，請詳細閱讀以下承諾：

1. 權限啟動與請求

當您主動啟動功能時，系統會先行跳出視窗請求存取您的相機權限。未經您的操作與授權，本程式不會擅自開啟相機或進行任何掃描行為。

2. 100% 裝置端本地處理

您的臉部辨識過程完全在您的個人行動裝置（On-Device）硬體中完成。所有的運算數據均不會上傳至我們的伺服器，確保生物辨識資訊不離開您的裝置。

3. 不儲存原始影像

系統不會儲存您的原始臉部照片或錄影檔案。掃描時會立即將臉部特徵轉換為一段加密的數學代碼（特徵向量），僅用於即時比對，且該代碼無法還原為原始人臉圖像。

4. 數據安全與刪除

辨識數據儲存於裝置的安全隔區（Secure Enclave）中，與作業系統其他部分隔離。當您關閉功能或卸載本程式時，相關特徵數據將同步被永久刪除。
