隱私聲明 (Privacy Statement)

為了保障您的帳戶安全並提供更便捷的體驗，本服務提供臉部辨識功能。在使用此功能前，請詳細閱讀以下關於隱私保護的承諾：

1. 權限啟動與請求

明確授權：

當您主動啟動臉部掃描功能時，系統會先行跳出視窗請求存取您的相機權限。

自主掌控：

未經您的操作與授權，本程式不會擅自開啟相機或進行任何掃描行為。您可以隨時在裝置設置中撤回此權限。

2. 100% 裝置端本地處理

不傳輸雲端：

您的臉部辨識過程完全在您的個人行動裝置（On-Device）硬體中完成。

物理隔離：

所有的運算數據均不會上傳至我們的伺服器或任何第三方雲端平台，確保生物辨識資訊不離開您的手掌心。

3. 不儲存原始影像

非影像化儲存：

系統不會儲存您的原始臉部照片或錄影檔案。

特徵碼加密：

掃描時會立即將臉部特徵轉換為一段加密的數學代碼（特徵向量），僅用於比對身分，且該代碼無法還原為原始人臉圖像。

4. 數據安全與刪除

硬體級保護：

辨識數據儲存於裝置的安全隔區（Secure Enclave / TEE）中，與作業系統其他部分隔離。

即時清除：

當您關閉臉部辨識功能或卸載本程式時，儲存於裝置端的相關特徵數據將同步被永久刪除。

To ensure your account security and provide a more convenient experience, this service offers facial recognition. Before using this function, please carefully read the following privacy commitments:

1. Permission Activation and Request

Explicit Authorization:

When you actively activate the facial scanning function, the system will first request access to your camera via a pop-up window.

Self-Control:

Without your operation and authorization, this program will not open the camera or perform any scanning behavior. You can revoke this permission at any time in your device settings.

2. 100% On-Device Local Processing

No Cloud Transfer:

Your facial recognition process is entirely completed on your personal mobile device (on-device) hardware.

Physical Isolation:

All processing data is not uploaded to our servers or any third-party cloud platforms, ensuring that biometric information never leaves your hands.

3. No Original Image Storage

Non-Image Storage:

The system will not store your original facial photos or video files.

Feature Encryption:

During scanning, facial features are immediately converted into an encrypted mathematical code (feature vector), used solely for identity verification. This code cannot be reconstructed from the original facial image.

4. Data Security and Deletion

Hardware-Level Protection:

Identification data is stored in the device's secure enclave (TEE), isolated from other parts of the operating system.

Instant Deletion:

When you disable the facial recognition function or uninstall this program, the relevant feature data stored on the device will be permanently deleted.
